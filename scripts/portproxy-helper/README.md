# Helpers

Набор небольших служебных скриптов.

## PortProxy Helper

`scripts/portproxy-helper.bat` — пакетный скрипт Windows для управления
правилами `netsh interface portproxy` (v4→v4) с дублированием правила v6→v4.

### Возможности
- добавление или обновление правила перенаправления (создаётся пара v4→v4 и v6→v4);
- перезапуск службы IP Helper для сброса существующих соединений;
- удаление отдельных правил или всех сразу;
- отображение текущей конфигурации PortProxy.

### Использование
1. Запустите `scripts/portproxy-helper.bat` в Windows. При необходимости
   скрипт перезапустит себя с правами администратора и закроет исходное окно.
2. Выберите пункт меню.
3. Следуйте подсказкам для настройки прокси.

IP‑адрес назначения по умолчанию можно изменить, отредактировав переменную
`DEFAULT_DEST_IP` в начале скрипта.
